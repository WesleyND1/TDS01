link para baixar o git
git-scm.com

Iniciar um novo projeto com git
git init

Add os arquivos que estâo prontos para serem commitados
git add <nome-arquivo>/ ou .

commit os arquivos no histórico
git commit -m "<Sua mensagem de commit>"

altera o ultimo commit
git commit --amend -m "<Sua mensagem de commit>"

mostra todos os commits
git log

mostra o que foi alterado na ramificaçâo
git diff

cria uma nova branch a partir da branch atual
git branch -b <nome-da-nova-branch>

listar a branch e mostrar a branch atual (que estamos)
git branch

muda para esta branch
git checkout <nome-branch>

git merge
merge de ramificaçâo(mescla a ramificaçâo)

configura o usuáio do github
git config --global user.name "<seu-nome>"

configura o email do github
git config --global user.email "<seu-email-cadastrado-no@git.com>"

mostra todas as configurações que temos já feito
git config --global --list 

mostra todas configurações que podemos fazer
git config --list

manda nossas alterações locais para o repositório remoto, para cada branch
git remote add origin <nome-da-branch>

manda nossas alteraçõeslocais para o repositório remoto
git push
git push origin <nome-da-branch>

pega as alterações do repositório remoto e joga para a nossa máquina
git pull
git pull origin <nome-da-branch>

atualiza o nosso histórico local de acordo com o histórico salvo la no repositório remoto
git fetch

copia (clona um repositório)
git clone <url-do-repositório>

copia (clona um repositório) e cria uma pasta com o clone dentro
git clone <url-do-repositório> <nome-da-pasta>

volta para o arquivo original, cancela todas as modificações feitas
git restore <nome do arquivo>
